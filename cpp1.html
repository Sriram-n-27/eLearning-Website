Introduction

C++ is a middle-level programming language developed by Bjarne Stroustrup starting in 1979 at Bell Labs.
C++ runs on a variety of platforms, such as Windows, Mac OS, and the various versions of UNIX.
This tutorial adopts a simple and practical approach to describe the concepts of C++.

Overview

  • C++ was developed by Bjarne Stroustrup starting in 1979 at Bell Labs in Murray Hill, New Jersey.
    • 

  • It is regarded as a middle-level language, as it comprises a 
combination of both high-level and low-level language features.
    • 

  • C++ is a superset of C, and that virtually any legal C program is a legal C++ program.
    • 

  • C++ fully supports object-oriented programming
    • 

  • C++ is being highly used to write device drivers and other software 
that rely on direct manipulation of hardware under realtime constraints.
    •
